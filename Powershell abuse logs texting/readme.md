W tym scenariuszu przeanalizowano uruchomienie PowerShell na hoście Windows 11 monitorowanym przez Wazuh. Telemetria została zebrana z Security log, **PowerShell** Operational oraz Sysmon. Test wykazał, że lokalne eventy 4688, 4103, 4104 i Sysmon ID 1 dają pełny obraz wykonania, natomiast widoczność w dashboardzie Wazuh zależy od reguł alertowania. W celu poprawy widoczności Sysmon Event ID 1 zastosowano custom rule, dzięki czemu możliwa była pełna analiza command line bezpośrednio w Wazuh.
